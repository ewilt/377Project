# Prince George’s County Health Finder

## User Guide 

PGHF is designed to facilitate the location of medical services throughout Prince George’s County by allowing the user to input their address, whether it is their current location, or that of their home, school, or workplace. The app will then find the nearest hospital to the inputted address, and return a map view of both the user’s and hospital’s location, as well as contact information for the facility.
